Commands


Command Usage
Announce <message> Bot says message with text to speech.
ggif <search query> Get random gif matching tags from Google.
Gif <image tags> Returns a random gif matching the tags passed.
idle [game] Sets bot status to idle.
Image <search query> Gets the top matching image from Google.
Leet <message> Converts boring regular text to 1337.
Log <log message> Logs message to bot console.
Myid Returns the user id of the sender.
Online [game] Sets bot status to online.
Ping Pong!
Rimage <search query> Gets a random image matching tags from Google.
Roll [# of sides] Roll one die with x sides, or multiple dice using d20 syntax. Default value is 10.
Say <message> Bot says message.
Talk <message> Talk directly to the bot.
Troll <user to troll> Trolls the @mentioned User!
Twitch <stream> Checks if the given stream is online.
Uptime Returns the amount of time since the bot started.
Urban <word> Looks up a word on Urban Dictionary.
Userid <user to get id of> Returns the unique id of a user. This is useful for permissions.
Version Returns the git commit this bot is running.
Wiki <search terms> Returns the summary of the first matching search result from Wikipedia.
Yesno Returns Yes or No.
Youtube <video tags> Gets Youtube video matching tags.
Play <search terms|URL> Plays the given video in the user's voice channel. Supports YouTube and many others: http://rg3.github.io/youtube-dl/supportedsites.html
Pause Pauses music playback.
Queue Prints the current music queue for this server.
Resume Resumes music playback.
Skip Skips to the next song in the playback queue.
Stop Disconnects JARVIS from all voice connections.
Volume <volume|volume%|volume dB> Set music playback volume as a fraction, a percent, or in dB.
Startpoll <choice a>, <choice b>, <choice c>... Starts a poll.
Endpoll Ends the current poll.
Viewpoll Returns the stats for the current poll.
Vote <name of option> Vote in the current poll.
